**OBJETIVO**

El presente trabajo tiene como objetivo simular las estructuras de datos estudiadas según silabo y sus variantes esta simulación será dada utilizando cualquier lenguaje de programación

Las estructuras de datos a ser presentadas en el simulador como mínimo son (pueden sugerirse otras) :

- Pila

- Cola

- Lista

- Arboles binarios


Simular la estructura de datos para mostrar las funciones de insertar y eliminar correspondientes:

- Mostrar en forma gráfica

- Mostrar explicación

Para realizar estas tareas se deberá usar en una página web
